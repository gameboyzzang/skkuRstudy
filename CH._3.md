벡터를 생성하는 c()함수


```r
metallica <- c('Lars','James','Jason','Kirl')
metallica
```

```
## [1] "Lars"  "James" "Jason" "Kirl"
```

벡터에서 특정 값 빼기


```r
metallica <- metallica[metallica != 'Jason']
metallica
```

```
## [1] "Lars"  "James" "Kirl"
```

벡터에 특정 값 추가


```r
metallica <- c(metallica, 'Rob')
metallica
```

```
## [1] "Lars"  "James" "Kirl"  "Rob"
```
